# Cron 定时任务配置

## 任务列表

| 时间 | 任务 | 说明 |
|------|------|------|
| 周一 20:00 | Coordination Agent 周拆解 | 提取章节，拆解4个微知识点，输出《本周地图》 |
| 周二至周五 19:00 | 生产次日微内容 | Researcher→Writer→Draw Agent 全流程 |
| 周二至周六 09:00 | 审核并生成发布清单 | Editor 校对红线，输出发布清单 |

## 注意事项

- 周二至周四：案例全部使用 NVIDIA/AMD/Intel
- 周五：可包含"行业多样性观察"，客观提及国内算力
- 周日：休息，不推送
- 审核重点：检查招聘话术、过度捆绑国产生态
